# bloodlink-website
BloodLink is a smart web platform that connects blood donors with recipients quickly and efficiently during emergencies. It enables users to search donors by blood group and location, ensuring fast access to life-saving support while promoting awareness about blood donation.


🩸 **College Project Spotlight: BloodLink – Emergency Blood Donation Network**

As part of my engineering journey, I developed **BloodLink**, a smart web platform aimed at solving a real-world problem—**connecting blood donors with patients during emergencies**.

🎯 **Project Objective:**
To create a reliable system that ensures quick access to blood donors while maintaining **donor safety and awareness**.

🚀 **Key Features:**
🌐 Multi-language support (English, Hindi, Marathi)
🧠 Medical eligibility & health validation system
🚨 Emergency blood request alerts
🧑‍🤝‍🧑 Donor registration & availability tracking
📊 Coordinator dashboard for managing requests
📚 Awareness section: Learn about blood donation, laws & health tips

💡 This project helped me apply concepts of **web development, user experience design, and problem-solving** to build something impactful for society.

❤️ **Goal:** Use technology to ensure that no life is lost due to shortage of blood.

👨‍💻 Tech Stack: HTML, CSS, JavaScript (with optional backend using Node.js/Firebase)

📌 Open to feedback and suggestions!
